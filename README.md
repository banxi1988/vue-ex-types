# vue-ex-types

vue 项目常用的一些辅助声明

# Install

```bash
npm i vue-ex-types -D
#  或 yarn add vue-ex-types -D
```

# How to Use

## add this module in your tsconfig.json

```json
{
  "compilerOptions": {
    "types": ["vue-ex-types"]
  }
}
```

## add implements to your mpvue page component

then you can get doc tip and autocomplete tip
